CCNAQuest · App Icon Preview

To export production icon PNG:

1. Open this file in Chrome or Safari
2. Right-click the large icon → Inspect, set element size to exactly 1024 × 1024
3. Or use the Puppeteer one-liner below:

npx puppeteer-cli screenshot --width 1024 --height 1024 icon-preview.html icon.png

Checklist before handing off to App Store Connect:
· Hat tip visible within top safe area (80% iOS clip circle) ✓
· No text or border baked in ✓
· Teal glow behind frog at 15–18% opacity ✓
· Background exactly #0e1a20 at edges (seamless transition to app) ✓

Android adaptive icon: Deliver frog-only (transparent bg) as assets/adaptive-icon.png. Background layer = #0e1a20 solid in app.json.
